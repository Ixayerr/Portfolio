# Портфоліо — Васильєв Богдан Анатолійович

Персональний портфоліо-сайт, створений у рамках лабораторної роботи №1 з курсу веб-розробки.

## Сторінки

- `index.html` — Головна сторінка з привітанням та ключовими навичками
- `about.html` — Детальна інформація про мене, освіта, таблиця навичок
- `contacts.html` — Контактна інформація та форма зворотного зв'язку

## Технології

- HTML5 (семантична розмітка)
- Git / GitHub
